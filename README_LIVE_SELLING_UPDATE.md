# TrustCart Complete Code with Live Selling Samples

This package restores the preferred marketplace dashboard based on the earlier home.html layout and adds TrustCart Live sample online-selling videos.

Included:
- Buyer and Seller platform only
- Back-to-school marketplace banner
- Search, categories, location targeting, quick action buttons
- TrustCart Live sample videos using provided YouTube Shorts IDs
- WELCOME10 first-time buyer discount code
- Seller-created discount codes
- Product photos and at least 3 products per category
- Trust Score, Green Score, protected seller location, pickup hub flow
- TrustPoints, autoship, checkout, order tracking, refund/protection pages
- FAQ, site map, policy pages
- Render + Supabase ready files

Run locally:
```bash
mvn spring-boot:run
```

Render/Supabase:
Use `database/supabase-setup.sql` if you want to reset and reload all tables and seed data, including WELCOME10.
